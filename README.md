## Chapter 1
### Style Sheets

#### External
``
    <link rel="stylesheet" href="css/style.css">
``

#### Internal
```
    <style>
        p {
            color: limegreen;
        }
    </style>
``` 

#### Inline
```
<p style="color:red">I am red</p>
```
